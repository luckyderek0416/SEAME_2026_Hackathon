# D-Racer 트러블슈팅 메모

## 차량이 전진 안 함 (스로틀 줄 때 "삐" 소리, 모터 안 돎)

**증상**: 조향(서보)은 되는데 스로틀을 줘도 구동 모터가 안 돌고 ESC에서 "삐" 소리.

**원인**: 스로틀 신호가 ESC의 **데드밴드(중립 근처 무시 구간)** 안에 머묾.
- `accel_ratio` 기본값 0.12에서는 스틱을 끝까지 밀어도 스로틀이 0.12뿐
- D3Racer 변환상 ≈ 1560us (중립 1500 + 60) → 대부분 ESC가 무시하는 ±50~100us 데드밴드 안

**해결**: **accel_ratio를 올림** (조이스틱 R1 버튼 또는 파라미터).
- 신호가 데드밴드를 넘어 모터가 구동됨
- 자율주행에선 decision의 `drive_throttle` / `slow_throttle`을 ESC가 도는 값 이상으로 설정해야 동일 효과

**관련 파라미터**:
- joystick: `R1`로 accel_ratio 증가 (수동주행)
- decision: `drive_throttle`(기본 0.18), `slow_throttle`(0.12) — ESC 데드밴드 넘는지 확인
- 후진은 코드상 막혀 있음(`throttle = max(0.0, ...)`)이라 전진만 테스트

**교훈**: 모터가 안 돌면 먼저 throttle 값을 데드밴드 위로 올려볼 것. 배터리/배선보다 이게 흔함.
