# 런 로그 — 2026-07-12 새벽~오전 (run30 ~ run48)

각 런마다 **직전 런과 달라진 것**(코드/파라미터/물리)과 **초 단위 타임라인**을 기록한다.
시간 표기: `vN` = 영상 기준 초 (v0 = 초록불 인식·출발, 오차 ±0.5s). 영상이 없거나
동기 불가한 런은 `G+N`(초록불 후 경과초) / `RA+N`(RA 진입 후 경과초)로 표기.
텔레메트리 원본: 노트북 `~/SEAME_telemetry/2026-07-12/`. 영상: Downloads/runNN.*

> ## ★ 최고 기록: **run47** — 무개입 전체 체인 완주
> 출발 → 흰 추종 → 분기(인코스) 진입 → 노란 접근로 → RA 진입 → 링 순환(구 약점
> 구간 자력 통과) → 개구부 재통과 → **게이트 탈출**(정지선+yaw 4.42) → 탈출로 추종.
> 죽어가는 7.8V 팩으로 달성. 유일한 흠: 탈출로 끝 노랑→흰 전환부 소실 직진
> 이탈(→ run48에서 조향-유지 폴백으로 수정).

## 이 구간에서 확립된 사실 (캘리 자산)

- **PWM 양자화**: 1틱 = 스로틀 0.0098 (`round(us/20000×4096)`). 0.16≡0.165(틱324),
  0.17≡0.174(틱325). 틱 경계 미만 변경은 하드웨어 무효.
- **속력 실측** (출발→래치 250cm): 0.19 기준 밴드별 49.5~62.8cm/s (±13%),
  +1틱당 +15~25%. 성공 런(54/59/60) 53~56cm/s.
- **링 경계**: 성공 최고 45.2cm/s(run55) / 진입 실패 run62. 적정 타깃 후보 ~40cm/s.
  링 속력 = 9129.72mm/랩타임.
- **배터리**: 무부하 8.0V(명목 ~80%)여도 부하 새그 ~1V — 내부저항 열화로 잔량과
  출력이 분리됨. 전압 회복 곡선 = 정지 감지 신호로 사용 가능.
- **가짜/진짜 정지선**: 링에 입구 2개(아래쪽 연결로=가짜, 우리 입구=진짜), 가로선
  시각적으로 동일. 기울기 부호 판별 불가 재확인(진짜가 양부호 다 냄). 유일 판별자
  = 링 위 위치 → 속도 프록시가 아닌 **군집 카운트**로 가야 함.

## 남은 문제 (이 구간 종료 시점)

1. **게이트 위치 판정의 속도 의존** — 군집 카운트(roundabout_exit_gates=2) 전환 승인 대기
2. **V-피드포워드 속도 고정** — 캘리 (V, 틱, 속력) 산점 수집 중 (배터리 로깅 가동)
3. 탈출 락 slow 캡 실차 미검증 (run61 이후 정상 발화 런 없음)
4. 정지 재킥 / battery_node 활성화 (printk 폭주 이력 가드 확인 후)
5. 배터리 교체 — 여전히 모든 것의 전제
